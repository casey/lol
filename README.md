# lol

## Usage

```
#!/usr/bin/env lol

<description of what you want the script to do>
```

## Installation

```
cargo install lol
```

## Features

### Inference Mode

Optionally, the script may be left blank, in which case Claude will attempt to
infer what to do based on the name of the script and its arguments.

## Examples

Please see the [bin](bin) directory.
